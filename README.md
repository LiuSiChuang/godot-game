# 🗡️ Лесной Охотник (Lesnoy Okhotnik)

## 📌 Описание проекта
Данный проект представляет собой 2D пиксельную игру в жанре приключения с элементами боя, разработанную на игровом движке Godot.

Игрок управляет персонажем, который перемещается по карте, сражается с врагами (например, кабанами), использует атаки и уклоняется от урона.

---

## 🎮 Основные возможности
- Движение персонажа (влево/вправо, прыжки)
- Боевая система (атаки, комбо-удары)
- Враги с искусственным интеллектом
- Система получения урона и отталкивания (knockback)
- Анимации персонажа и врагов
- Система состояний (State Machine)

---

## 🛠️ Используемые технологии
- Godot Engine 4.x
- Язык программирования: GDScript
- 2D Pixel Art графика
- TileMap для построения уровней
